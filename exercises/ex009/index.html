Trabalhando com Listas

Listas Ordenadas

Ordered List ou Lista Ordenada são listas enumeradas, usa-se a tag <ol>. Podemos mudar a apresentação da lista para 5 tipos diferentes: lista numérica, alfabética maiúscula e minúscula ou numérica romana maúscula ou minúscula. Para isso usa-se <ol type=""> no valor adiciona 1, A, a, I ou i.

  1. Acordar
  2. Ligar para o joão
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Voltar para casa
  7. Jantar
  8. Dormir

Listas não Ordenadas

Unordered List ou Lista não ordenada é a bullet list, sinalizado por ponto, quadrado, estrela... Usa-se a tag <ul>.

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Lista de Definição

Definition List ou Lista de Definição é a lista de dicionário, que contém termo e significado do termo. Usa-se a tag <dl>.

HTML
Linguagem de marcação para criação de conteúdo para um site.
CSS
Linguagem de marcação para criação de design de um site.
JavaScript
Linguagem de programação para criação de interatividade de um site.
